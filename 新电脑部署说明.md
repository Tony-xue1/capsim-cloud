# 🖥️ 新电脑部署说明

## 前置要求

- **Node.js**（必需）
  - 下载：https://nodejs.org/zh-cn
  - 选择 LTS 版本（推荐 v18 或更高）
  - 安装时全部默认即可

---

## 部署步骤

### 方法一：使用一键安装脚本（推荐）

1. 双击运行 `一键安装.bat`

2. 按提示操作：
   - 脚本会自动安装依赖
   - 提示输入 API Key 时，粘贴你的 DeepSeek API Key

3. 安装完成后，双击 `启动服务.bat` 启动

4. 打开浏览器访问：**http://localhost:8080**

---

### 方法二：手动部署

```bash
# 1. 安装依赖
npm install

# 2. 创建配置文件
# 在项目根目录创建 config.json，内容如下：
# {
#   "apiKey": "你的 DeepSeek API Key"
# }

# 3. 启动服务
node server.js

# 4. 访问
# 打开浏览器：http://localhost:8080
```

## 获取 API Key

1. 访问 https://platform.deepseek.com/
2. 注册/登录账号
3. 点击左侧 **"API Keys"**
4. 点击 **"Create new secret key"**
5. 复制生成的 Key（格式：`sk-` 开头）

## 常见问题

### Q: 提示"npm 不是内部或外部命令"
**A:** Node.js 未安装或未添加到 PATH，请重新安装 Node.js

### Q: 启动后无法访问 http://localhost:8080
**A:** 检查端口 8080 是否被占用，或修改 `server.js` 中的端口号

### Q: 上传 PDF 后提示错误
**A:** 检查 `config.json` 中的 API Key 是否正确，以及网络是否能访问 DeepSeek API
